\initial{O}\textbf{pen Motion Planning Library, is a software library which we have been tasked with agentifying. The benefit of agentifying software components, is the software components become self aware. These simple agentified software components allow for the development of more intelligent software agents. The goal is for a software agent which can dynamically choose the best configuration of it's available agentified components to accomplish a given task. In the OMPL, there exist Planner components; our end goal is to have an agent which can dynamically choose the best planner to use based on the task requirements.}
